# YYB_iOSComponents

1. ##### **YYBCategory** **:**  日常开发分类文件

   ```
   pod 'YYB_iOSComponent', :subspecs => ["Category"]
   ```

   ```
   pod 'YYB_iOSComponent', :subspecs => ["Router"]
   ```
